# plugin-blog-hao
**让你的博客变得更好**

> 为你的博客和站点增加一些更使用的功能，比如文本复制追加功能，单页面和文章集成 swiper幻灯片、PDF等文件展示功能！总之，只要有需求，就有功能！

## 插件正在开发中

如果您看到了这个插件，有什么好的建议和想法可以加入如下渠道进行交流。

## 目前已经完成的功能
- 图片轮播功能（四种风格）
- 哔哩哔哩视频嵌入
- PDF文件嵌入
- 文本复制追加功能
- 全站失色功能
- 中英文符号自动空格，段落首行缩进功能
- 幻灯片展示
